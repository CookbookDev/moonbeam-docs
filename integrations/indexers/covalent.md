# Getting Started with the Covalent API

![The Graph on Moonbeam](/images/thegraph/thegraph-banner.png)

## Introduction

Covalent provides a unified API to bring full transparency and visibility to assets across all blockchain networks. Simply put, Covalent offers a single API that allows you to pull detailed, granular blockchain transaction data from multiple blockchains with no code. The unified Covalent API allows you to create entirely new applications or augment existing ones without configuring or maintaining infrastructure. Covalent supports Moonbase Alpha and plans to support the PolkaDot and Moonriver deployments.
